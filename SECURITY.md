# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in arch-guard, please report it privately instead of opening a public issue.

**Email**: pabloalvarezgonzalez4190@gmail.com

Do not use the issue tracker for security vulnerabilities — we want to fix them before they're publicly disclosed.

## What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

We'll acknowledge your report within 48 hours and work on a fix. We'll keep you informed of progress.
